# How to deploy?

`Workflow Method.`

- Fork or Import This Repo

### Add These in Secrets

- `HEROKU_EMAIL`
- `HEROKU_API_KEY`
- `HEROKU_APP_NAME`

Upload Config.env (Either Zmirror or Weebzone) in This Repo.

  >**NOTE:** If you are using Mirror & Clone Functions, Upload all relevent files in here.
